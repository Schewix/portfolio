A personal knowledge base built with Next.js, MDX, and automated deployments. The goal was to create a high-performance writing system that still feels like a product.

## Impact
- Templated MDX workflows with automatic previews.
- Consistent content structure for fast reuse.
- Time-to-publish reduced to minutes.

## Lessons learned
- Strong information architecture makes content scale.
- Static builds can still feel dynamic with the right UX.
- A light design system keeps the site maintainable.

## Stack
- Next.js App Router
- MDX content pipeline
- Vercel preview deployments
